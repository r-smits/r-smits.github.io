GlobeTrotter - GlobeTrotter v. 0.01

Description
This projects aims to create an accurate mercator projection of earth. The aim is to add interactive elements to the globe in later releases. Currenlty working on increasing rendering efficiency as the projection matrix multiplications on > 5000 points is creating latency issues.

To achieve smoother animation, V. 0.02 will add dynamic line simplification through application of the visvalingam-whyatt algorithm. V. 0.03 will add linear interpolation and click event handling to support interactive rotation.
