Algochart - Algochart v. 0.05

Description
This projects aims to use simple mathematical concepts and javascript to avoid using existing libraries as much as possible. Coded without external libraries to champion new challenges and learn as much as possible in the process.

GlobeTrotter - GlobeTrotter v. 0.04

Description
This projects aims to create an interactive globe, coded from scratch.

v.0.02:
- Line simplification with visvalingam-whyatt algorithm.
- Bug: Topology is not conserved so I will implement dedup and a shared-edge algorithm.
- Adding linear interpolation after that and click event handling to support interactive rotation.

v.0.03:
- Added JSON parsing, specifically topoJSON parsing with httprequest.
- Introduced shared arcs. Reverses delta-encoding.
- Option to render topoJSON as a flat map or a globe.

v.0.04:
- Dynamic line simplification - visvalingam-whyatt alteration.
- Bug fix: Maintenance of topology through shared arcs.
- The mesh can now be dynamically simplified without altering rotation.

v.0.05: (pendng):
- Quaternions and SLERP to enable point interpolation and replace rotational matrices.
- Test Cordic for rotational matrices versus quaternions.
- Introducing intersection and winding order for user - input.
